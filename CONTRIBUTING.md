# Contributing

Thank you for your interest in contributing to `skill-release-publish`.

## Development setup

```bash
git clone https://github.com/metyatech/skill-release-publish.git
cd skill-release-publish
```

## Submitting changes

1. Fork the repository and create a feature branch.
2. Add or update tests for any changed behavior.
3. Open a pull request with a clear description of the change.

## Scope

This skill covers **release preparation, package publishing, and post-publish verification**. Please keep PRs scoped to release workflow concerns.
